# 📸 PhotoVault

PhotoVault is a modern and responsive photography gallery website built using React and Vite. It allows users to browse beautiful images with category filters, search functionality, a lightbox preview, and dark mode.

## 🚀 Live Demo

https://photovault-nk3u.vercel.app

## ✨ Features

- Responsive design
- Hero section
- Image gallery
- Search photos
- Category filters
- Lightbox image preview
- Dark mode
- About section
- Contact section
- Smooth scrolling
- Modern UI

## 🛠️ Built With

- React
- Vite
- JavaScript
- CSS
- HTML

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/Prachi-cipher/photovault.git
```

Go to the project folder:

```bash
cd photovault
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

## 🚀 Build

```bash
npm run build
```
